# Project Uas Praktikum
### Management Playlist Musics
Program ini menggunakan deque untuk management penyimpanan datanya(in-memory) dan struct untuk bentuk datanya. Lalu program ini memiliki multi role: user dan admin.
Program ini Menggunakan functional programming. 

### Feature Yang tersedia
1. Buat Playlist
2. Update Playlist
3. Lihat Playlist
4. Tambah Musik
5. Update Musik
6. Delete Musik
7. Lihat Musik
7. Play Musik
8. Register
8. Login
9. Logout
